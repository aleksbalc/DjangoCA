Welcome to the Certification Authority webpage

This application was created to ensure secure data transfer necessary for registering a node in the MQTT network between nodes KS and N.


Thesis task content:

1. Familiarization with the KGR key generation and renewal system for IoT network nodes.

2. Development of the Authorization Center (CA) concept for the KGRD system meeting the requirements:

- Graphic application interface for CA management;

- Secure data exchange between CA and the key server of the KGRD system;

- Secure transfer of authentication data for clients of the KGRD system.

3. Development of a tool demonstrator.

4. Development of scenarios for presenting the demonstrator.

5. Preparation of a report on the implementation of individual scenarios.


This is a part of an engineering-level thesis project created in 2024.
